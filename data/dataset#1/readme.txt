The "rawData" folder contains the data files in dataset#2.

Before running our provided QoS prediction approaches, you need to run "dataProcessor.m" to generate the traning data and testing data. You can run it under the default configurations, or modify the script by your own.

You can download the entire dataset at: http://www.cse.cuhk.edu.hk/~jmzhu/dataset/wsrec_dataset1.zip
